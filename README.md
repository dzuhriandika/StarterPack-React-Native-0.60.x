## Getting Started

- Clone this repository
> git clone https://github.com/dzuhriandika/StarterPack-React-Native-0.60.x.git
- CD to StarterPack
> cd ./StarterPack-React-Native-0.60x
- Install module
> npm i --save
- Rename this project
> react-native-rename newname
- Run this project
> react-native run-android
